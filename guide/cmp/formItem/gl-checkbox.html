# Checkbox 多选框

# 常规用法

新增 checkboxs 属性，传递数据。
新增 isCheckAll 属性，当设置 isCheckAll 为 true 时，提供全选框。

<template>
  <div>
    <gl-checkbox :checkboxs="checkboxs1" v-model="checkedValue1" isCheckAll @change="handleChange"></gl-checkbox>
    <gl-checkbox :checkboxs="checkboxs2" v-model="checkedValue2" disabledSingle="['苹果', '草莓']"></gl-checkbox>
    <gl-checkbox :checkboxs="checkboxs3" v-model="checkedValue3" checkButton></gl-checkbox>
  </div>
</template>

<script lang="ts" setup>
import { ref } from 'vue'

const checkboxs1 = [
  {name: 'cityA', id: '北京'},
  {name: 'cityB', id: '上海'},
  {name: 'cityC', id: '重庆'}
]
const checkedValue1 = ref(['北京'])

const checkboxs2 = [
  {name: 'fruitA', id: '香蕉'},
  {name: 'fruitB', id: '苹果'},
  {name: 'fruitC', id: '西瓜'},
  {name: 'fruitD', id: '草莓'}
]
const checkedValue2 = ref(['香蕉', '苹果'])

const checkboxs3 = [
  {name: 'colorA', id: '红'},
  {name: 'colorB', id: '蓝'},
]
const checkedValue3 = ref([])

const handleChange = (value: Array<string | number | boolean>) => {
  console.log('change')
}
</script>

<style lang="scss" scoped>
.gl-checkbox{
  margin: 20px;
}
</style>```
属性名说明类型可选值默认值
checkboxs数据，v-for、lable等属性失效，通过 labelKey 控制 labelArray必须
v-model绑定值Array必须
nameKeycheckbox 中对象的键名string'name'
labelKeycheckbox 中对象的键名string'id'
isCheckAll是否需要全选框booleantrue/falsefalse
checkButton按钮样式多选框booleantrue/falsefalse
needTooltip（todo）是否需要文字提示，仅当checkButton为false生效booleantrue/falsefalse
isShow（todo）checkbox 中对象的键名，控制是否展示文字提示，仅当checkButton为false和needTooltip为true生效string-
disabledSingle被禁用的项Array-
