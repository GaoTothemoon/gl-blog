# Input 输入框

# 常规用法

支持 el-input 所有用法， 新增提示框，当设置tooltip时，提示框出现。

<template>

    <gl-input v-model="inputValue3" type="password" placeholder="请输入" @click="myClick($event)"></gl-input>
    <gl-input v-model="inputValue4" placeholder="请输入" @keyup.enter="submit"></gl-input>

    <gl-input v-model="inputValue6" placeholder="请输入网址" @input="myInput">

</template>
