# Radio 单选框

# 常规用法

新增 radios 属性，传递数据。 使用radioButton控制单选框样式。

<template>
  <div>
    <gl-radio :radios="radios1" v-model="radioValue1"></gl-radio>
    <gl-radio :radios="radios1" v-model="radioValue1" radioButton></gl-radio>
  </div>
</template>

<script lang='ts' setup>
import { ref } from "vue";

const radios1 = [
  {name: '西瓜', id: 'a'},
  {name: '香蕉', id: 'b'},
  {name: '苹果', id: 'c'}
]
let radioValue1 = ref('c')

</script>

<style lang="scss" scoped>
.gl-radio{
  margin: 16px;
}
</style>```
属性名说明类型可选值默认值
radios数据，v-for、lable等属性失效，通过 labelKey 控制 labelobject[]必须
v-model绑定值string / number / boolean必须
nameKeycheckbox 中对象的键名string'name'
labelKeycheckbox 中对象的键名string'id'
radioButton按钮样式多选框booleantrue/falsefalse
disabledSingle被禁用的项Array-
