# 属性

属性名说明类型可选值默认值
header卡片头部标题string-
needCheck控制是否出现单选框，仅当设置 header 属性或设置 header 插槽时生效booleanfalse
disabledCheck控制单选框是否可选booleanfalse

# 新增插槽

插槽名说明
header （块级元素出问题）于卡片头部
default默认插槽，于卡片体
operate（todo）于卡片头部提供下拉框

# 事件

事件名说明参数类型
@change通过此事件判断单选框的值string / number / booleanFunction
