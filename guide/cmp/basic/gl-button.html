# Button 按钮

# 常规用法

支持 el-button 所有用法， 新增气泡框用法。

<template>
  <el-row>
    <gl-button type="primary" @click="handleClick">按钮</gl-button>
    <gl-button type="success" sort='常规' icon="Edit" />
    <gl-button type="primary" sort='run' :popConfirm='true' @confirm="handleConfirm"/>
    <gl-button type="warning" sort='query' :popConfirm='true' popWidth="300" />
    <gl-button type="info" sort='config' :popConfirm='true' popTitle="自定义弹窗title" />
    <gl-button type="primary" sort='backup' :popConfirm='true' popTitle="自定义弹窗title" icon="Edit" />
  </el-row>
</template>

<script lang='ts' setup>
const handleConfirm = ():void => {
  console.log('confirm')
}
const handleClick = ():void => {
  console.log('click')
}
</script>

<style lang="scss" scoped>
.gl-button {
  margin: 12px;
}
</style>```

# 属性

属性名说明类型可选值默认值
sort按钮文体string见下方 SORTS 表-
popConfirmContent Cellbooleanfalse
popTitleContent Cellstring请确认执行此操作？
popWidthContent Cellnumber200
teleportedContent Cellbooleantrue
